# Skeleton Templates

These are v2 Phase 4 base templates for skeleton-writer. They are not manuscript prose. Fork the relevant type template into a project under `04_skeleton/section_skeleton.md` after Human Gate 2.

Files:

- `typeA_skeleton.md` for empirical asset pricing.
- `typeB_skeleton.md` for real estate empirics.
- `typeC_skeleton.md` for forecasting.
- `typeD_skeleton.md` for interpretable ML in finance.
- `typeE_skeleton.md` for conceptual or institutional papers.

Each template defines section order, typical paragraph maps, one claim per paragraph, exhibit reference, and expected literature category. Figure 1 hook placement is explicit in the introduction.
